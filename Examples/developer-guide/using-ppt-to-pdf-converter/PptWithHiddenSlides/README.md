# PPT with Hidden Slides

This example demonstrates how to convert ppt with hidden slides to pdf using GroupDocs.Conversion.LowCode.

## Overview

This console application shows how to convert ppt with hidden slides to pdf.

## Purpose

This example demonstrates the **ppt with hidden slides workflow**:
- Convert PPT with hidden slides to PDF
- Basic file input/output handling
- Straightforward conversion process

## Files

### Input Files
- `business-plan.docx` - Source document for conversion

### Output Files
- `business-plan.pdf` - Converted output document

## Running the Example

1. Ensure you have .NET 6.0 or later installed
2. Set your GroupDocs license keys as environment variables:
   - `GD_PUBLIC_KEY`
   - `GD_PRIVATE_KEY`
3. Navigate to this directory
4. Run the example:
   ```bash
   dotnet run
   ```

## Code Overview

The conversion logic in Program.fs demonstrates:
- License setup from environment variables
- Convert PPT with hidden slides to PDF
- Basic conversion workflow
- File output

## Requirements

- .NET 6.0 or later
- GroupDocs.Conversion.LowCode package
